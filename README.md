# Vertex -  LLM/GraphLLM工具
## 功能
* 支持本地Ollma部署获取模型(chatbox)
* 支持通过API方式调用外部模型(chatbox)
## 环境要求

- Python 3.8+
- Ollama (https://ollama.com)

## 安装

1. 安装 Ollama:
   - 访问 https://ollama.com/download
   - 下载适用于您系统的安装包并安装

2. 克隆仓库：
```bash
git clone git@github.com:ashione/vertex.git
cd vertex


3. 本地安装:
```shell
pip install -r requirements.txt
pip install -e . 
```

4. 执行vertex启动
```shell
vertex
```
